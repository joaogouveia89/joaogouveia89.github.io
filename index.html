It features Google Maps integration, offline support, contact management, and a clean architecture with robust error handling. Number classificator app(MNIST Dataset)

Number Classifier App (MNIST Dataset) is an educational Android application that showcases on-device machine learning by recognizing handwritten digits using a custom-trained deep learning model. The model was trained by the developer during a machine learning bootcamp, and the repository includes the full training notebook used in the course. Built with Kotlin and TensorFlow Lite, the app allows users to draw digits and get real-time predictions, highlighting the power of edge AI on Android devices.

UFC Data Analysis Dashboard

UFC Data Analysis Dashboard is an end-to-end data analysis project that explores over 30 years of UFC fights. The work includes descriptive statistics, comparisons analytics and performance metrics to identify winning patterns, impact of age, fighting styles, and historical rivalries. The results are presented in a dashboard interactive and in the repository notebooks, combining Python, Pandas, Jupyter and interactive visualizations.

Project Euler Solutions is a personal repository containing solutions to various mathematical and programming challenges from Project Euler. Each problem is solved using clean, efficient code with a focus on readability and mathematical insight. Sorting Comparasion App

Sorting Comparison App is a simple yet effective Android application that benchmarks two classic sorting algorithms — Bubble Sort and Quick Sort — on a list of 3,000 randomly generated items. The app measures and displays the execution time of each algorithm directly in the UI, providing a clear visual comparison. Built with Kotlin, Jetpack Compose, and Coroutines, the app features a modern, responsive interface and reactive programming principles. This project was designed as an educational experiment to explore algorithm efficiency and UI responsiveness in real-world Android environments.

GitHub
Traffic Simulation

A C++ simulation project developed during the Udacity C++ Nanodegree (Concurrency module), where vehicles navigate a city with intersections controlled by traffic lights. This project focuses on implementing a thread-safe communication system between traffic lights and vehicles using modern C++ concurrency features like mutexes, locks, and message queues. The simulation includes dynamic traffic light behavior with randomized phase changes, ensuring safe and realistic vehicle flow. The repository includes the complete implementation along with build instructions and project tasks as defined in the course.
